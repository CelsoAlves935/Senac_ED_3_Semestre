# 🏗️ Estrutura de Dados com Java - Senac

Este repositório reúne as implementações e estudos práticos realizados durante a disciplina de **Estrutura de Dados** no curso de Tecnologia em Análise e Desenvolvimento de Sistemas.

O objetivo deste material é documentar o aprendizado sobre a organização, gerenciamento e armazenamento de dados de forma eficiente.

## 🚀 Conceitos e Estruturas Implementadas
* **Vetores e Arrays:** Manipulação básica e busca de elementos.
* **Listas Ligadas (Linked Lists):** Implementação de nós e encadeamento manual.
* **Pilhas (Stacks) e Filas (Queues):** Conceitos de LIFO (Last In, First Out) e FIFO (First In, First Out).
* **Algoritmos de Ordenação:** (Ex: Bubble Sort, Selection Sort ou Quick Sort).
* **Recursividade:** Lógica para resolução de problemas complexos.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java (JDK 17 ou superior).
* **IDE Sugerida:** VS Code, IntelliJ IDEA ou Eclipse.

## 📂 Organização do Repositório
Os exercícios estão organizados por pastas temáticas, facilitando a consulta de algoritmos específicos:
- `/arrays`: Exercícios com vetores estáticos.
- `/listas`: Implementações de listas dinâmicas.
- `/pilhas-filas`: Exemplos práticos de uso de stacks e queues.
